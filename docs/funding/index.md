# Funding

## Mål

Bruke første betalte pilot eller sterk kundeinteresse som grunnlag for Innovasjon Norge.

## Funding track

| Milepael | Status |
| --- | --- |
| Finn riktig Innovasjon Norge-lop | Ikke startet |
| Definer pilotbevis | Pagaar |
| Lag enkel budsjett/BOM | Ikke startet |
| Samle kundeinteresse | Pagaar |
| Skriv finansieringshistorie | Ikke startet |

## Prinsipp

Funding-historien skal bygges rundt reelt kundebehov, ikke bare teknologi.
