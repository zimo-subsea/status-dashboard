# Innovasjon Norge

## Hva vi trenger

- Et tydelig kundeproblem.
- Betalt pilot eller sterk skriftlig interesse.
- Enkel og troverdig produktretning.
- Budsjett/BOM.
- Forklaring på hvorfor Zimo bør få støtte nå.

## Bevis fra pilot

Piloten bør gi:

- Kundens problem formulert med deres ord.
- Praktisk brukssituasjon.
- Hva kameraet gjorde nyttig.
- Hva som må forbedres.
- Om kunden vil betale videre eller anbefale løsningen.
