# Produkt

## Kort forklart

Zimo bygger et robust undervannskamerasystem for oppdrett, vanninfrastruktur og industriell inspeksjon.

## Første produktpakke

- Ett undervannskamera.
- En topside gateway.
- Live view.
- Opptak.
- Enkel support.
- Ett tydelig pilot-use-case.
- Kort oppsummering etter pilot.

## Designkrav

| Område | Status |
| --- | --- |
| Trykk | 100 bar som designkrav |
| Kabel | 15 meter er nok for første bruk |
| Kamera | GMSL-basert retning |
| Gateway | Topside Jetson/SBC i første fase |
| Undervannsdel | Holdes enkel og robust |
